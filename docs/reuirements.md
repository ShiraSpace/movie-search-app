# Take-Home Interview: MovieList App

Time limit: 2 hours

## What to build

A simple web app where a user can:

- Search for movies (powered by OMDB API)
- Click a movie to view its details
- Add/remove movies from a personal watchlist
- View the saved watchlist

## API

Use the OMDB API: http://www.omdbapi.com/
Your API key: 

Example search: `http://www.omdbapi.com/?s=inception&apikey=<<API_KEY>>`
Example detail: `http://www.omdbapi.com/?i=tt1375666&apikey=<<API_KEY>>`
